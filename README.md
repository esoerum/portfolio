## Instructions
Create .env file in backend folder with the following code:
```
DATABASE_URL = dev.db
PORT = 3000
FRONTEND_URL = http://localhost:5173
```
To install required packages and seed database from root:  
```
pnpm run setup
```
To start frontend and backend:  
```
pnpm run dev
```
